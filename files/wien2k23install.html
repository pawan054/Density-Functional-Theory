Wien2k_23.2 Installation
  • Requirements
  • Install requirements
  • Download Wien2k tar file
  • Install Wine2k
  • Set up user configuration
    •
    In this tutorial, you will learn how to install Wien2k 23.2 version using gfortran in your Linux operating system. Since Wien2k is mostly written in Fortran language, you need to install some related libraries and packages. Install following packages:
    Install prerequisites
  • sudo apt-get update
  • sudo apt-get upgrade
  • sudo apt-get install gfortran
  • sudo apt-get install grace
  • sudo apt-get install gnuplot
  • sudo apt-get install csh
  • sudo apt-get install tcsh
  • sudo apt-get install tk
  • sudo apt-get install build-essential
  • sudo apt-get install ghostscript
  • sudo apt-get install c++
  • sudo apt-get install perl6
  • sudo apt-get install octave
  • sudo apt-get install gfortran
  • sudo apt-get install gcc
  • sudo apt-get install bc
  • sudo apt-get install make
  • sudo apt-get install cmake
  • sudo apt-get install awk
  • sudo apt-get install vi gedit emacs
  • sudo apt-get install okular evince xpdf
  • sudo apt-get install python3
  • sudo apt-get install gfortran
  • sudo apt-get install python3-pip
  • pip3 install numpy
  • sudo apt-get install libblas-dev
  • sudo apt-get install liblapack-dev
  • sudo apt-get install libopenblas-dev
  • sudo apt-get install libvips
    •  Install FFTW3
  • Download FFTW 3.3.10 from website https://www.fftw.org/fftw-3.3.10.tar.gz
  • Look for it in Download folder, create folder software in home, and move it here i.e.,
    cd Downloads
    mv fftw-3.3.10.tar.gz home/software
    cd home/software
  • tar -xvf fftw-3.3.10.tar.gz
  • cd fftw-3.3.10
  • ./configure --prefix=/opt/fftw3
  • make
  • sudo make install
    •  Install Xcryden
  • sudo apt install xcrysden
  • copy the path and add to .bash.rc file i.e.,
    which xcrysden
    gedit ~/.bashrc
  • at the bottom of .bashrc, copy following lines:
    xcy_top=usr/bin/
    xcy_bottom=/tmp
    export xcy_top xcy_bottom
    •  Install Wien2k
  • Download the Wien2k_23.2.tar file
  • tar -xvf wien2k WIEN2K_23.2.tar
  • gunzip *.gz
  • chmod +x ./expand lapw
  • ./expand lapw
  • ./siteconfig lapw
    • In new version of wien2k i.e. 24, add Real libraries=/usr/lib/x86_64-linux-gnu/libopenblas.so.0 while in older version, Real libraries=-lblas -llapack -lpthread
  • ./userconfig lapw
