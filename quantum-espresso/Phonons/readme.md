## How to calculate phonons at individual q-points
By Pawan Joshi <br/>
<hr/>
Here, you will learn how to calculate phonons frequencies at at individual irreduciable q-points. For instance when you want to run phonon at 2 2 1 q-mesh, then depending on the symmetry of crystal you will get q-points (i.e. in case of hexagonal 2 q-points) at which you are supposed run phonons separately. By this way you can control phonons at each point and make calculations smoother and faster. 
<br/>
You can do this either manually or use script wich is recommended. I have provided the scripts under directory-phonons
